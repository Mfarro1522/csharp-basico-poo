# C# Básico y Programación Orientada a Objetos

Este repositorio documenta mi proceso de aprendizaje en C#, enfocado en dominar la sintaxis del lenguaje, explorar el ecosistema .NET y familiarizarme con las formas de construir aplicaciones, desde consola hasta escritorio con Windows Forms (WinForms).

## Descripción del Curso

- Sintaxis de C# (variables, tipos de datos, operadores)
- Estructuras de control de flujo (condicionales y bucles)
- Manejo de excepciones (try-catch-finally)
- Clases, objetos y POO con la sintaxis de C#
- Interfaces y clases abstractas en C#
- Colecciones del ecosistema .NET (List, Dictionary, Arrays)
- Manejo de archivos con librerías .NET
- Exploración de librerías y herramientas del ecosistema .NET
- **Aplicaciones de escritorio con Windows Forms (WinForms)** 🖥️

## Objetivos

- Aprender la sintaxis de C# viniendo de Java
- Explorar el ecosistema .NET y sus librerías
- Familiarizarme con las formas de construir aplicaciones en .NET
- Aplicar los conocimientos de POO ya adquiridos en Java con la sintaxis de C#
- Construir aplicaciones de escritorio Windows con WinForms
